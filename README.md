## plms
个人生活管理系统
## 后台框架
Spring + Springboot + Mybatis Plugs + shiro + redis + token + AOP
## 前端框架
vue + vuex + iview + axios
## 登录功能
shiro+token+ redis + JWT生成token令牌实现单点登录
## 通过AOP和队列实现数据对接
通过AOP实现数据源切换，通过队列和线程池实现数据对接
## 通过AOP实现产品记录状态日志
通过AOP实现产品状态日志的记录
## mybatis两种代码生成器
mybatis自动生成器 + Mybatis Plugs 自动生成器
## 实现controller单元测试
